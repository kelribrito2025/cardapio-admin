# Comparação de Layout: Login vs Onboarding

## Diferenças Identificadas

### Login
- Logo: gap-3, mb-8 lg:mb-6
- Header: mb-8, título text-2xl, descrição text-gray-500 (sem tamanho especificado)
- Form: space-y-5
- Input: h-14, pl-12, text-base
- Button: h-14, text-base
- Padding do container direito: p-8

### Onboarding
- Logo mobile: gap-3, mb-4
- Header: mb-2, título text-lg, descrição text-xs
- Form: space-y-4
- Input: h-12, pl-10, text-sm
- Button: h-12, text-sm
- Padding do container direito: p-6

## Ajustes Necessários
1. Aumentar altura dos inputs de h-12 para h-14
2. Aumentar padding left dos inputs de pl-10 para pl-12
3. Aumentar tamanho do texto de text-sm para text-base
4. Aumentar espaçamento do form de space-y-4 para space-y-5
5. Aumentar padding do container de p-6 para p-8
6. Ajustar tamanho do header de text-lg para text-2xl
7. Ajustar margin bottom do header de mb-2 para mb-8
8. Aumentar altura do botão de h-12 para h-14
